A small React + TypeScript app built with Vite and Tailwind CSS.

## Setup (Quick)

1) Install
```bash
npm install
```

2) Run (dev)
```bash
npm run dev
```
Open the shown URL (usually http://localhost:5173/).

3) Build (prod)
```bash
npm run build
```

4) Preview build
```bash
npm run preview
```
